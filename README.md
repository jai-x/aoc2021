# aoc2021


#### requirements

Ensure `asdf` version manager is installed correctly:
```shell
# install erlang plugin
$ asdf plugin-add erlang

# install elixir plugin
$ asdf plugin-add elixir

# install erlang and elxir
$ asdf install
```

#### running

```shell
# compile
$ mix escript.build

# run
$ ./aoc2021 --day <num>
```
